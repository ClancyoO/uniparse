# UniParse · 全能视频解析站 - 设计文档

## 1. 项目概述

### 1.1 目标
创建一个**工业赛博风**的全能视频解析网页，支持国内外主流视频网站的视频解析、播放和下载。用户只需粘贴视频链接，即可获取高清视频地址，支持选择画质、下载封面和字幕。

### 1.2 核心价值
- **一站式解析**：支持 B站、YouTube、抖音、腾讯视频等国内外主流视频平台
- **快速下载**：直接获取视频直链，无需等待缓冲
- **画质自选**：列出所有可用画质供用户选择
- **附加资源**：支持下载封面和字幕
- **酷炫体验**：像素侠客终端动画，增强交互趣味性

### 1.3 技术方案
- **前端**：纯 HTML + CSS + JavaScript（工业赛博风视觉）
- **后端**：Vercel Serverless Functions（Node.js 环境，无需管理服务器）
- **部署方式**：推送代码到 GitHub → 连接 Vercel 自动部署
- **国内可访问**：Vercel 生成的 `.vercel.app` 域名国内可直接访问
- **临时出现“演示模式”**：配置后端后自动转为真实解析

### 3.2 数据流

1. **输入阶段**：用户粘贴链接或输入搜索关键词
2. **识别阶段**：前端解析 URL，识别视频平台
3. **请求阶段**：前端调用 Vercel API（`/api/parse`），传递视频 URL
4. **解析阶段**：Vercel Serverless Function 根据平台类型选择对应解析策略，去目标网站抓取视频数据
5. **返回阶段**：Vercel API 返回视频信息（画质列表、封面、字幕、视频地址）
6. **展示阶段**：前端展示结果，用户选择画质后播放或下载

#### 4.2.2 侠客终端动画设计

**侠客形象**（CSS 像素动画）：
- 角色：像素风侠客（斗笠、披风、长剑）
- 动作序列：
  1. **待命状态**：侠客站立，剑在鞘中
  2. **拔剑状态**：解析开始，侠客拔剑，剑光闪烁
  3. **挥剑状态**：每完成一个任务（解析、获取封面、获取字幕），侠客挥剑横斩
  4. **收剑状态**：全部完成，侠客收剑归鞘，任务清单显示 ✅

**日志文案风格**（口语化）：
- "叮！收到一个链接，侦查一下..."
- "是个B站视频！BVxxxxxx"
- "看我一剑斩断！发起解析请求！"
- "米人解析源：一击命中！"
- "封面？我的！字幕？也是我的！"
- "剑光闪过，任务全灭 ✅"

**任务斩灭动画**：
- 任务完成时，对应文字被剑光划过（CSS 动画）
- 文字变为删除线 + 火花粒子效果
- 侠客动画与任务完成同步

| 结果展示 | `index.html` | 视频信息 + 画质选择 + 播放器 |

### 5.2 Vercel Serverless API

**文件**：`api/parse.js`

**功能**：
- 接收前端请求（视频 URL）
- 根据 URL 识别平台
- 调用对应平台的解析函数
- 返回解析结果（视频地址、画质列表、封面、字幕）

**平台解析策略**：

| 平台 | 解析方式 |
|------|----------|
| B站 | 请求 `api.bilibili.com/x/web-interface/view` 获取视频信息，请求 `x/player/playurl` 获取播放地址 |
| YouTube | 请求视频页面，解析页面中的视频流数据 |
| 抖音/TikTok | 请求移动端 API，获取视频流 |
| 腾讯视频 | 请求页面，解析 m3u8 播放列表 |

## 6. 部署与使用

### 6.1 Vercel 部署

1. 在 GitHub 上创建仓库，上传所有文件
2. 访问 [https://vercel.com](https://vercel.com) 并用 GitHub 账号登录
3. 点击 **"Add New Project"** → 选择刚才的 GitHub 仓库
4. 默认设置即可（Framework Preset 选 **Other**）
5. 点 **"Deploy"**，几秒后自动完成
6. 获取分配的 `.vercel.app` 域名（如 `uniparse.vercel.app`）

### 6.2 前端使用

- **本地测试**：直接双击 `index.html` 或用 `python3 -m http.server` 启动（API 调用会走演示模式）
- **公网使用**：部署到 Vercel 后自动可用，所有功能正常运作

## 7. 安全与隐私

- **无数据存储**：前端不存储用户的视频链接或任何个人信息
- **HTTPS**：确保所有请求通过 HTTPS
- **CORS 处理**：通过 Vercel API 后端代理请求，绕过浏览器跨域限制
- **无追踪**：不收集用户行为数据
